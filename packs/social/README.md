# Social Media Pack
15 premium skills for social media monitoring, analytics, and content management.

## Skills Included

1. **twitter-monitor.yaml** - Monitor Twitter/X keywords and hashtags, track mentions and trending topics in real-time.
2. **reddit-search.yaml** - Search Reddit posts and comments, retrieve subreddit content, and analyze discussions.
3. **rss-reader.yaml** - Read and parse RSS/Atom feeds, extract articles, and monitor feed updates.
4. **hackernews.yaml** - Fetch trending stories from Hacker News, search stories, and get user activity.
5. **github-trending.yaml** - Fetch trending repositories on GitHub, search repositories, and get repository details.
6. **arxiv-search.yaml** - Search arXiv for academic papers, get paper details, and monitor new publications.
7. **product-hunt.yaml** - Get daily top products from Product Hunt, search products, and track product launches.
8. **youtube-info.yaml** - Get YouTube video information, search videos, and extract channel details.
9. **linkedin-post.yaml** - Create and manage LinkedIn posts, analyze engagement, and schedule content.
10. **medium-articles.yaml** - Search and retrieve popular Medium articles, get author details, and track topics.
11. **devto-articles.yaml** - Fetch trending articles from Dev.to, search articles, and get user publications.
12. **stackoverflow.yaml** - Search Stack Overflow questions, get answers, and retrieve user activity.
13. **news-aggregator.yaml** - Aggregate news from multiple sources, search by topic, and get top headlines.
14. **blog-monitor.yaml** - Monitor blog updates, track RSS feeds, and get notifications for new posts.
15. **social-analytics.yaml** - Analyze social media metrics, track engagement, and generate performance reports.

## Features

- All skills use HTTP/shell tools to fetch real data
- Support for multiple platforms and APIs
- Bilingual descriptions (EN/ZH)
- Premium quality with comprehensive parameters
- Real-time monitoring capabilities

## Requirements

- Some skills require API keys (Twitter, YouTube, LinkedIn, etc.)
- Python 3 with feedparser for RSS parsing
- jq for JSON processing
- curl for HTTP requests

## Usage

Each skill can be triggered by natural language commands related to its functionality. For example:
- "Monitor Twitter for #AI keywords"
- "Search Reddit for machine learning posts"
- "Get trending GitHub repositories in Python"
