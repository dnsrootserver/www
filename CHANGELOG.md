# Changelog

## 1.0.0 (2026-04-18)

### Features
- Core YAML-based skill engine
- 10 built-in tools (shell, http, read_file, write_file, log, output, transform, prompt, set_variable, condition)
- 8 built-in skills (web-search, url-reader, system-info, code-review, github-repo-info, weather, crypto-price, hello-world)
- CLI with commands: init, run, chat, list, validate, doctor, examples
- Jinja2 template support in all params
- Regex and pattern trigger matching
- Conditional step execution
- Variable management
- JSON output mode
- Rich terminal UI
